Add Resources

Swift Craft Launcher supports automatically downloading mods, resource packs (or texture packs), shaders, and data packs from CurseForge and Modrinth, and adding them to your game.

Adding resource packs, data packs, and shaders follows the same process as installing mods below: enter the "Resource Library", switch to the corresponding tab (Resource Packs, Data Packs, or Shaders), search, and click "Install".

Add Mods

Important: Vanilla Minecraft does not support mods. If you want to install mods, make sure you have properly installed a mod loader.

Step 1: Enter Resource Library

In the "Game List" on the left, select the game for which you want to install mods, then click the top button to enter the "Resource Library".

Step 2: Search and Install Mods

Select the "Mods" option at the top, use the search bar on the right to find the mod you want to add. Once you find the mod, click "Install".

Step 3: Handle Dependencies

Some mods may display an interface as shown. This is because the mod requires other mods as dependencies to run. Don't worry, after clicking "Download All", the launcher will automatically install them for you. After the mod installation is complete, click the top button to return to "Installed".

Step 4: Confirm Installation

When the mod you just downloaded appears on the "Installed" page, the mod has been installed.

Add Modpacks

Step 1: Search Modpack

In the "Resource List" on the left, select "Modpacks". In the top search bar, search for the modpack you want to add, then click "Install".

Step 2: Configure Modpack

Select "Game Version" and "Modpack Version", name the version, then click "Download".

Step 3: Wait for Installation to Complete

The launcher will automatically install Java and the modpack for you.

Step 4: Complete Installation

The installed modpack will appear in the "Game List" on the left.

Add Servers

Swift Craft Launcher can add server resources from the Resource List directly into a game instance. After the action completes, the launcher writes the server entry into the standard Minecraft servers.dat file inside that instance's profile directory.

Step 1: Open the Server Resource Dialog

In the "Resource List", switch to the server resource list, then click "Add" on the target server resource. A detail dialog will open; for server resources, both the dialog title and the primary button will show "Add Server".

Step 2: Choose a Game Instance

The dialog first loads the server details and the list of compatible game instances. The launcher automatically filters out any instances that already contain the same server address, so the list only shows instances where this server can still be added. Select the target instance, then click "Add Server".

Step 3: Save to the Server List

The launcher parses the server address from the resource details and uses the resource title as the server name, then writes the result into the target instance's servers.dat. After that, the server will appear in that instance's server list.

This feature depends on the server address being available in the resource details. If the launcher cannot parse a valid address, adding the server will fail and an invalid address error will be shown.
