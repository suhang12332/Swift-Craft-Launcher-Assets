新增資源

Swift Craft Launcher 支援從 CurseForge 和 Modrinth 自動下載模組、資源包（或紋理包）、著色器和資料包，並將它們新增到遊戲中。

新增資源包、資料包、著色器與下方安裝模組流程相同：進入「資源庫」，切換到對應的標籤（資源包、資料包或著色器），搜尋後點擊「安裝」。

新增模組

【重要】Vanilla Minecraft 不支援模組。如果您想安裝模組，請確保您已正確安裝了模組載入器。

步驟 1：進入資源庫

在左側的「遊戲列表」中，選擇您要為其安裝模組的遊戲，然後點擊頂部按鈕進入「資源庫」。

步驟 2：搜尋並安裝模組

選擇頂部的「模組」選項，使用右側的搜尋欄查找您要新增的模組。找到模組後，點擊「安裝」。

步驟 3：處理依賴項

某些模組可能會顯示如圖所示的介面。這是因為模組需要其他模組作為依賴項才能運行。不用擔心，點擊「下載所有」後，啟動器會自動為您安裝它們。模組安裝完成後，點擊頂部按鈕返回「已安裝」。

步驟 4：確認安裝

當您剛剛下載的模組出現在「已安裝」頁面上時，表示模組已安裝。

新增模組包

步驟 1：搜尋模組包

在左側的「資源列表」中，選擇「模組包」。在頂部搜尋欄中，搜尋您要新增的模組包，然後點擊「安裝」。

步驟 2：配置模組包

選擇「遊戲版本」和「模組包版本」，命名版本，然後點擊「下載」。

步驟 3：等待安裝完成

啟動器將自動為您安裝 Java 和模組包。

步驟 4：完成安裝

已安裝的模組包將出現在左側的「遊戲列表」中。

新增伺服器

Swift Craft Launcher 支援將資源列表中的伺服器資源直接加入遊戲實例。完成後，啟動器會把伺服器資料寫入該實例 profile 目錄下的 Minecraft 標準 servers.dat 檔案。

步驟 1：開啟伺服器資源視窗

在左側進入「資源列表」，切換到伺服器資源列表後，點擊目標伺服器資源上的「新增」。此時會彈出詳情視窗；對於伺服器類型的資源，彈窗標題和底部主按鈕都會顯示為「新增伺服器」。

步驟 2：選擇要加入的遊戲實例

彈窗會先載入伺服器詳情與相容的遊戲實例清單。啟動器會自動過濾掉已經包含相同伺服器地址的實例，因此清單中只會顯示目前仍可加入該伺服器的遊戲實例。選擇目標實例後，點擊「新增伺服器」。

步驟 3：寫入伺服器列表

啟動器會從該伺服器資源的詳情資訊中解析伺服器地址，並以資源標題作為名稱，寫入目標實例的 servers.dat。完成後，您可以在對應遊戲實例的伺服器列表中看到它。

此功能依賴伺服器資源詳情中的地址資訊。如果詳情裡沒有可識別的伺服器地址，啟動器將無法加入，並提示地址無效。
