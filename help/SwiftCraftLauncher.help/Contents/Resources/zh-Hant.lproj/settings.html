啟動器設置

Swift Craft Launcher 提供了多種配置選項，讓您自定義使用體驗。

通用設置

語言選擇

選擇應用語言。更改後需要重啟應用才能生效。

支援的語言：阿拉伯語 (ar)、丹麥語 (da)、德語 (de)、英語 (en)、西班牙語 (es)、芬蘭語 (fi)、法語 (fr)、印地語 (hi)、義大利語 (it)、日語 (ja)、韓語 (ko)、挪威語 (nb)、荷蘭語 (nl)、波蘭語 (pl)、葡萄牙語 (pt)、俄語 (ru)、瑞典語 (sv)、泰語 (th)、土耳其語 (tr)、越南語 (vi)、簡體中文 (zh-Hans)、繁體中文 (zh-Hant)。

主題選擇

選擇主題模式：系統、淺色或深色。

啟動器工作目錄

設置啟動器工作目錄。啟動器管理的所有遊戲檔案都將放在此資料夾中。可以重置為預設值。

【重要】更改工作目錄後，原始路徑中的遊戲將不再出現在「遊戲列表」中。建議轉移或重新下載它們。最好在首次打開啟動器時更改工作目錄。

並發下載數量

設置並發下載數量，範圍：1-64。更高的並發性可以提高下載速度，但也會帶來更大的性能開銷。

遊戲設置

預設API源

選擇預設的資料來源。可用選項：Morinth 和 CurseForge。

AI崩潰分析

啟用/停用AI崩潰分析功能。此功能需要先成功配置AI設置。

預設記憶體分配

設置全域預設記憶體範圍（Xms-Xmx）。

遊戲資源資訊

顯示快取檔案數量和總大小（唯讀）。

遊戲高級設置

Java 路徑

自定義Java可執行檔案路徑，可重置為預設值。

以下是 MC 版本及其最低要求的 Java 版本：

  • 1.12 - Java 8

  • 1.17 - Java 16

  • 1.18 - Java 17

  • 1.20.5 - Java 21 (64-bit)

垃圾回收器

選擇垃圾回收器類型，根據Java版本自動過濾可用選項。

可用的垃圾回收器選項：

  • G1GC - Java 7+

  • ZGC - Java 11+

  • Shenandoah - Java 12+

  • Parallel - Java 1.0+

  • Serial - Java 1.0+

性能優化預設

選擇性能優化級別：

  • 停用 - 不啟用任何優化

  • 基礎 - 啟用基本優化

  • 平衡 - 啟用記憶體和執行緒優化（推薦）

  • 最大 - 啟用所有優化包括Aikar Flags（僅G1GC可用）

記憶體範圍

設置Xms-Xmx記憶體範圍（512 MB - 最大記憶體）。

自定義JVM參數

手動輸入JVM參數（與垃圾回收器和性能優化互斥）。

環境變數

設置遊戲運行時的環境變數。

AI設置

AI提供商選擇

選擇AI服務提供商（支援多個提供商）。

API金鑰

可顯示/隱藏的API金鑰輸入框。

Ollama地址

僅在選擇Ollama時顯示，預設：http://localhost:11434

AI頭像URL

設置AI頭像的URL地址，支援預覽。

玩家設置

【註】玩家設置功能正在開發中。請關注更新。
