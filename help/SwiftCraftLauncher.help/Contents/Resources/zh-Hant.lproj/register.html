登入帳戶

Swift Craft Launcher 支援三種主要的登入方式：離線模式、Microsoft 官方登入和第三方認證伺服器登入。

離線模式

無需網路驗證。輸入任意用戶名即可開始遊戲。僅適用於單人模式或已停用線上認證的伺服器。

步驟 1：新增玩家

在啟動器中點擊「新增玩家」。

步驟 2：選擇離線模式

選擇右上角的「離線」選項，輸入您想要的玩家名稱，然後點擊「建立帳戶」。

步驟 3：確認新增

當您的玩家資訊出現在啟動器左下角時，帳戶已成功新增。

Microsoft 登入

使用您的官方 Microsoft 帳戶進行認證。這是存取所有官方 Minecraft 功能的唯一方式。您必須擁有已購買的 Minecraft 副本才能使用此方法。

步驟 1：新增玩家

在啟動器中點擊「新增玩家」。

步驟 2：選擇 Microsoft 登入

選擇右上角的「Microsoft」選項，然後點擊「開始登入」。

步驟 3：瀏覽器登入

點擊允許打開瀏覽器，然後按照網頁上的說明登入您的 Microsoft 帳戶並授權 Swift Craft Launcher。

步驟 4：確認新增

返回啟動器並確認新增。

步驟 5：完成登入

當您的玩家資訊出現在啟動器左下角時，帳戶已成功新增。

第三方登入

使用第三方 Minecraft 認證伺服器登入，例如 LittleSkin、MUA 或 Ely.By。登入完成後，您可以選擇該伺服器回傳的角色，並在啟動遊戲時自動接入對應的認證與皮膚服務。

步驟 1：新增玩家

在啟動器中點擊「新增玩家」。

步驟 2：選擇第三方伺服器

切換到「第三方」登入類型，先選擇認證伺服器預設，再點擊「開始登入」。

目前內建支援 LittleSkin、MUA 和 Ely.By 三個常見服務。

步驟 3：瀏覽器授權

啟動器會打開瀏覽器，跳轉到對應第三方認證伺服器的授權頁面。請依照網頁提示登入並授權 Swift Craft Launcher。

步驟 4：選擇角色

授權完成後，啟動器會自動讀取該帳號下的角色清單。

如果伺服器回傳多個角色，您可以在介面中切換並選擇目前要新增的角色。

步驟 5：完成新增

當所選角色出現在啟動器左下角時，第三方帳戶已成功新增。

啟動遊戲時，啟動器會自動注入所需認證元件，以連接該第三方伺服器提供的認證與皮膚服務。
