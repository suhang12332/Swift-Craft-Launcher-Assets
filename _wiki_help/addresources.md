# Add Resources

Swift Craft Launcher supports automatically downloading mods, resource packs (or texture packs), shaders, and data packs from CurseForge and Modrinth, and adding them to your game.

Adding resource packs, data packs, and shaders follows the same process as installing mods below: enter the "Resource Library", switch to the corresponding tab (Resource Packs, Data Packs, or Shaders), search, and click "Install".

# Add Mods

<div class="Alert">

*Important:* Vanilla Minecraft does not support mods. If you want to install mods, make sure you have properly installed a mod loader.

</div>

<div id="mod-step1" class="Task">

## Step 1: Enter Resource Library

<div id="aria-mod-step1" class="TaskBody" role="region" aria-hidden="false">

In the "Game List" on the left, select the game for which you want to install mods, then click the top button to enter the "Resource Library".

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-1_1.png" alt="Resource Library">
</figure>

</div>

</div>

<div id="mod-step2" class="Task">

## Step 2: Search and Install Mods

<div id="aria-mod-step2" class="TaskBody" role="region" aria-hidden="false">

Select the "Mods" option at the top, use the search bar on the right to find the mod you want to add. Once you find the mod, click "Install".

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-1_2.png" alt="Mod search">
</figure>

</div>

</div>

<div id="mod-step3" class="Task">

## Step 3: Handle Dependencies

<div id="aria-mod-step3" class="TaskBody" role="region" aria-hidden="false">

Some mods may display an interface as shown. This is because the mod requires other mods as dependencies to run. Don't worry, after clicking "Download All", the launcher will automatically install them for you. After the mod installation is complete, click the top button to return to "Installed".

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-1_3.png" alt="Dependency installation">
</figure>

</div>

</div>

<div id="mod-step4" class="Task">

## Step 4: Confirm Installation

<div id="aria-mod-step4" class="TaskBody" role="region" aria-hidden="false">

When the mod you just downloaded appears on the "Installed" page, the mod has been installed.

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-1_4.png" alt="Installed mods">
</figure>

</div>

</div>

# Add Modpacks

<div id="modpack-step1" class="Task">

## Step 1: Search Modpack

<div id="aria-modpack-step1" class="TaskBody" role="region" aria-hidden="false">

In the "Resource List" on the left, select "Modpacks". In the top search bar, search for the modpack you want to add, then click "Install".

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-6_1.png" alt="Modpack search">
</figure>

</div>

</div>

<div id="modpack-step2" class="Task">

## Step 2: Configure Modpack

<div id="aria-modpack-step2" class="TaskBody" role="region" aria-hidden="false">

Select "Game Version" and "Modpack Version", name the version, then click "Download".

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-6_2.png" alt="Modpack configuration">
</figure>

</div>

</div>

<div id="modpack-step3" class="Task">

## Step 3: Wait for Installation to Complete

<div id="aria-modpack-step3" class="TaskBody" role="region" aria-hidden="false">

The launcher will automatically install Java and the modpack for you.

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-6_3.png" alt="Modpack installation">
</figure>

</div>

</div>

<div id="modpack-step4" class="Task">

## Step 4: Complete Installation

<div id="aria-modpack-step4" class="TaskBody" role="region" aria-hidden="false">

The installed modpack will appear in the "Game List" on the left.

<figure>
<img src="https://raw.githubusercontent.com/suhang12332/Swift-Craft-Launcher-Assets/refs/heads/main/imagebed/addresources/3-6_4.png" alt="Modpack complete">
</figure>

</div>

</div>
